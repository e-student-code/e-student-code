🐧 Installing Firefox developer (Works in any linux system)



Mozilla firefox - Solus OS


Download Firefox Edition in the official website:


Click here After the download is done, open the terminal in Downloads where the file is.


Open your terminal and extract the file: 

            
                sudo tar xjf firefox*.tar.bz2

   tar - extract or create a archive file

   xjf - x instructs tar to extract the contents of the firefox*.tar.bz2 archive;

     j instructs tar to decompress the archive using the bzip2 compression;

     f represents the filename of the archive file.


Extraction done, so now lets move the firefox folder to /opt/

            
                sudo mv firefox /opt/firefox-developer-edition

   mv – command moves the firefox directory to the /opt directory.

   /opt/firefox-developer-edition - destination directory where firefox is moved to. mv creates the firefox-developer-edition directory if it doesn’t exist



Create a symbolic link

            
                sudo ln -s /opt/firefox-developer-edition/firefox /usr/local/bin/firefox-developer-edition

All done, we can launch the firefox developer. if need, update or reboot your system.
